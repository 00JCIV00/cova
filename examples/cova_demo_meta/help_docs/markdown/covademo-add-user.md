# add-user
__[covademo](./covademo.md)__ > __add-user__

A demo sub command for adding a user.

___

## Usage
```shell
USAGE
    add-user [ --fname |  --lname |  --age |  --admin |  --ref-ids ]
```

## Arguments
### Options
- __fname__:
    - `-f, --fname <fname (text)>`
    - The 'fname' Option.
- __lname__:
    - `-l, --lname <lname (text)>`
    - The 'lname' Option.
- __age__:
    - `-a, --age <age (u8)>`
    - The 'age' Option.
- __admin__:
    - `-A, --admin <admin (toggle)>`
    - The 'admin' Option.
- __ref-ids__:
    - `-r, --ref-ids <ref-ids (u8)>`
    - The 'ref-ids' Value.
### Values
- __id__ (u16)
    - The 'id' Value.
