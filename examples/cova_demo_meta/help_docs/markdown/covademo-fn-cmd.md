# fn-cmd
__[covademo](./covademo.md)__ > __fn-cmd__

A demo sub command made from a function.

___

## Usage
```shell
USAGE
        fn-cmd```

## Arguments
### Values
- __int__ (i32)
    - The first Integer Value for the fn-cmd.
- __string__ (text)
    - The first String Value for the fn-cmd.
- __byte_array__ (u8)
    - A 6-Byte Array for fn-cmd
