# union-cmd
__[covademo](./covademo.md)__ > __union-cmd__

A demo sub command made from a union.

___

## Usage
```shell
USAGE
    union-cmd [ --int |  --str ]
```

## Arguments
### Options
- __int__:
    - `-i, --int <int (i32)>`
    - The first Integer Value for the union-cmd.
- __str__:
    - `-s, --str <str (text)>`
    - The first String Value for the union-cmd.
### Values
- __union-uint__ (u8)
    - The 'union-uint' Value.
- __union-str__ (text)
    - The 'union-str' Value.
