# fn-cmd
__[covademo](./covademo.md)__ > __fn-cmd__

A demo sub command made from a function.

___

## Usage
```shell
USAGE:
    fn-cmd "val-00 (i32)" | "val-01 (text)"
    fn-cmd 

```

## Arguments
### Values
- __val-00__ (i32)
    - No description. (Descriptions cannot currently be generated from Function Parameters.)
- __val-01__ (text)
    - No description. (Descriptions cannot currently be generated from Function Parameters.)
